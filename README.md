# Stardew Valley Translations German/Deutsch

Collection of my German translations for Stardew Valley mods. / Alle meine deutschen Übersetzungen von Stardew-Valley-Mods.

Installation: Datei "de.json" in den Ordner "i18n" des jeweiligen Mods kopieren. Sollte die Installation mal anders funktionieren, gebe ich das beim jeweiligen Mod an.
